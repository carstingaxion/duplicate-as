
## plugin.php

- [`duplicate_as_after_duplicate`](duplicate_as_after_duplicate) Fires after a post has been duplicated
- [`duplicate_as_excluded_meta_keys`](duplicate_as_excluded_meta_keys) Filters the list of meta keys to exclude from duplication
- [`duplicate_as_featured_image`](duplicate_as_featured_image) Filters the featured image ID to copy
- [`duplicate_as_meta_value`](duplicate_as_meta_value) Filters the meta value before adding it to the duplicate post
- [`duplicate_as_post_data`](duplicate_as_post_data) Filters the post data before creating a duplicate
- [`duplicate_as_taxonomies`](duplicate_as_taxonomies) Filters the taxonomies to copy during duplication
- [`duplicate_as_taxonomy_terms`](duplicate_as_taxonomy_terms) Filters the terms to copy for a specific taxonomy
